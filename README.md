## 팀 규칙

- 주 1~2회 회의 진행하기
- 주 단위 목표를 정하고 실행하기 (+@@@!)
- 원하는 추가 기능이 있다면 즉시 공유하기
- 서로의 영역을 구분짓지 않고 목표 달성을 위해 협업 지향하기


## 프로젝트 개요
정보처리기사 준비하는 학생들의 실력을 향상시키기 위해 개발하였습니다.
